# 🎬 Anime Data Analysis Project

## 📌 Overview
This project analyzes an anime dataset to understand patterns in ratings, popularity, genres, and audience engagement.

## 🔧 Work Done
- Data Cleaning (handled missing values)
- Exploratory Data Analysis (EDA)
- Genre Analysis using split and explode
- Year-wise trend analysis
- Feature Engineering (popularity category, episode category, engagement score)
- Visualization using graphs

## 📊 Key Insights
- Anime quality is consistently high over time
- Most anime have 12–24 episodes
- Popularity and ratings are not always the same
- Anime is mainly targeted towards teenagers (PG-13)
- Engagement varies across anime

## 🛠 Tools Used
- Python
- Pandas
- Seaborn
